# GoREST
Тестирование проекта GoREST с помощью Postman.

Объект - **Comments**.

Действия: **Get List** (получение списка постов), **Create Post** (создание поста).
1) Анализ технической документации.
2) Составление интеллект-карты.
3) Составление запросов.
4) Тестирование.

Запросы: **Get List**, **Create Post**, **Get Post by ID**, **Update Post**, **Delete Post**.
